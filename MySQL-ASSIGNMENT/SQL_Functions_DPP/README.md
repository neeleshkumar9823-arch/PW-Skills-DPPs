# 🗄️ SQL Functions – Practice Assignment

📌 A **MySQL-based SQL assignment** focused on applying **core SQL functions, aggregation, grouping, and analytical queries**
using **World** and **Sakila** sample databases.

🎯 Designed to demonstrate **practical SQL problem-solving skills** required for **Data Analyst, Data Science, and Backend**
entry-level roles.

---

## 🎯 Objectives

✅ Apply aggregate functions like `COUNT`, `SUM`
✅ Use grouping and filtering with `GROUP BY` and `HAVING`
✅ Perform analytical queries on real-world datasets
✅ Derive business insights from relational databases
✅ Write clean, readable, interview-ready SQL

## 📊 Final Output (Expected State)

📁 After execution, the queries return:

* Country with **maximum population**
* Total **city population per country**
* **Top 3 population-dense countries**
* Most active **customers and rental months**
* **Daily and store-wise revenue insights**
* Logical validation of **empty result sets** where applicable

⚠️ Some queries (e.g. customers with exactly 5 payments) may return **no rows**
depending on the dataset — this is **expected and correct behavior**.
