# 🔗 SQL Subqueries – Practice Assignment

📌 A **MySQL-based SQL assignment** focused on applying **subqueries**
to analyze **employees, departments, and sales performance**
using a **corporate business dataset**.

🎯 Designed to demonstrate **nested query logic and analytical thinking**
required for **Data Analyst and Data Science** entry-level roles.

---

## 🎯 Objectives

✅ Understand and apply SQL subqueries  
✅ Use scalar, correlated, and nested subqueries  
✅ Perform employee and department-level analysis  
✅ Analyze sales performance using subqueries  
✅ Filter data using aggregate subqueries  
✅ Write clean, structured, interview-ready SQL  

---

## 🛠️ Tech Stack

🗄️ **Database:** MySQL  
📜 **Language:** SQL  
⚙️ **MySQL Version:** 8.0+  
📈 **Level:** Beginner → Advanced

1️⃣ Open **MySQL Workbench** or any MySQL client  
2️⃣ Create tables: `employees`, `departments`, `sales`  
3️⃣ Insert dataset values as provided

## 📊 Final Output (Expected State)

📁 After execution, the queries return:

* Employees earning above average salary  
* Departments with highest average salaries  
* Employees with and without sales  
* Sales performance insights  
* Department-wise sales totals

* Salary vs sales comparisons
