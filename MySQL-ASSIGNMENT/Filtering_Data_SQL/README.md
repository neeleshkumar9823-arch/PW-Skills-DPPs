
# 🗄️ Sales Transactions SQL – Practice Assignment

📌 A **MySQL-based SQL assignment** focused on applying **core SQL filtering, logical operators, aggregation, grouping, and analytical queries**
using a **Sales Transaction dataset**.

🎯 Designed to demonstrate **practical SQL problem-solving skills** required for **Data Analyst and Data Science**
entry-level roles.

---

## 🎯 Objectives

✅ Apply filtering using `WHERE`, `AND`, `OR`  
✅ Use sorting with `ORDER BY`  
✅ Perform aggregation using `COUNT`, `SUM`, `AVG`  
✅ Group and filter aggregated data with `GROUP BY` and `HAVING`  
✅ Analyze sales performance across **categories** and **regions**  
✅ Write clean, readable, interview-ready SQL  

---

## 🛠️ Tech Stack

🗄️ **Database:** MySQL  
📜 **Language:** SQL  
⚙️ **MySQL Version:** 8.0+  
📈 **Level:** Beginner → Intermediate  

---

## 📂 What’s Inside

| 🔹 Section | 📄 Description |
|-----------|---------------|
| 🔍 Basic Filtering | Queries using `WHERE` to filter transactions |
| 🔗 Logical Operators | Combined conditions using `AND` |
| 📅 Date-Based Filtering | Transaction analysis using date conditions |
