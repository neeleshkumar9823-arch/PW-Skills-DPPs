# 🗄️ Introduction to SQL & Databases – Practice Assignment

📌 A **MySQL-based SQL practice repository** covering database creation, data manipulation, filtering, and constraints.  
🎯 Built to demonstrate **core SQL fundamentals** for **Data Analyst, Data Science, and Backend** entry-level roles.

---

## 🎯 Objectives

✅ Create and manage databases & tables  
✅ Perform CRUD operations  
✅ Filter data using `WHERE`, `AND`, `DISTINCT`  
✅ Enforce data integrity with constraints  

---

## 🛠️ Tech Stack

🗄️ **Database:** MySQL  
📜 **Language:** SQL  
📈 **Level:** Beginner → Intermediate  

---

## 📂 What’s Inside

| 🔹 Section | 📄 Description |
|----------|---------------|
| 🏗️ Database Setup | Create & select database |
| 🧱 Table Creation | Define schema |
| ➕ Data Insertion | Insert multiple records |
| 🔍 Querying Data | `SELECT`, `WHERE`, `DISTINCT` |
| ✏️ Data Modification | `INSERT`, `UPDATE`, `DELETE` |
| 🔐 Constraints | PK, UNIQUE, NOT NULL, CHECK |

---

## ▶️ How to Run

1️⃣ Open **MySQL Workbench** (or any MySQL client)  
2️⃣ Create a new SQL tab  
3️⃣ Run `introduction_to_sql.sql`  
4️⃣ Execute queries step-by-step to observe results  

---

## 📊 Final Output (Expected State)

📁 After all operations, the `employees` table reflects **cleaned and validated employee records** with applied constraints.
