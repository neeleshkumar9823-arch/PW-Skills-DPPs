# Excel Pivot Table Assignment 📊

This assignment demonstrates how to use **PivotTables in Excel** for summarizing, filtering, sorting, grouping, and visualizing data.  
Each question is listed with the exact task and a short explanation of the steps taken to solve it.

---

## ✅ Questions & Steps

### Q1: Summarizing Revenue and Profit
**Question:** Create a PivotTable to show Revenue and Profit grouped by Region, Month, and Category.  
**Steps Taken:** Added Revenue and Cost fields, created a calculated Profit field, grouped by Region, Month, and Category to display totals.

---

### Q2: Filtering Data (Show only Electronics)
**Question:** Filter the PivotTable to show only Electronics category.  
**Steps Taken:** Dragged Category into Filters area, selected *Electronics* from dropdown → table shows only Electronics sales.

---

### Q3: Sorting Revenue in Descending Order
**Question:** Sort regions by Revenue from largest to smallest.  
**Steps Taken:** Right‑clicked Revenue values → Sort → Largest to Smallest → regions/categories reordered by revenue.

---

### Q4: Grouping Data by Month
**Question:** Group Transaction Dates into Months.  
**Steps Taken:** Added Date field to Rows → Right‑clicked → Group → selected Months → monthly sales trends displayed.

---

### Q5: Adding a Calculated Field (Profit = Revenue – Cost)
**Question:** Add a calculated field for Profit.  
**Steps Taken:** PivotTable Analyze → Fields, Items & Sets → Calculated Field → formula `=Revenue - Cost` → Profit column added.

---

### Q6: Using Slicers for Interactive Filtering
**Question:** Insert a Slicer for Salesperson.  
**Steps Taken:** Inserted Slicer → selected Salesperson → slicer buttons allow quick filtering by individual salesperson.

---

### Q7: Identifying the Best-Selling Product
**Question:** Find the product category with highest revenue.  
**Steps Taken:** Placed Product Category in Rows, Revenue in Values → sorted descending → top category identified.

---

### Q8: Applying Conditional Formatting (Revenue > $1,500)
**Question:** Highlight revenue values greater than $1,500.  
**Steps Taken:** Selected Revenue column → Conditional Formatting → New Rule → greater than 1500 → applied highlight color.

---

### Q9: Creating a Pivot Chart (Revenue by Product Category)
**Question:** Create a PivotChart to compare revenue by product category.  
**Steps Taken:** Inserted PivotChart → chose Column Chart → visual comparison of revenue across categories.

---

### Q10: Refreshing the Pivot Table (Update with New Data)
**Question:** Update PivotTable when new transactions are added.  
**Steps Taken:** Ensured dataset range includes new rows (or converted to Table) → PivotTable Analyze → Refresh → table updated.

---

## 🎯 Learning Outcomes
- Summarized and aggregated data with PivotTables.  
- Applied filters, sorting, and grouping for deeper insights.  
- Created calculated fields for custom metrics.  
- Used slicers and conditional formatting for interactive analysis.  
- Built PivotCharts for visualization.  
- Refreshed PivotTables to include new data.
